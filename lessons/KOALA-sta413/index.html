If Z follows a standard normal distribution — written Z ∼ N(0, 1) — then Z² follows a chi-square distribution with 1 degree of freedom: Z² ∼ χ²(1).

The transpose of a vector flips it from a column to a row (or vice versa). If X = (X₁, X₂)′ is a column vector, then X′ = [X₁   X₂] is a row vector.

An eigenvalue λ of a matrix A is a number that satisfies: Av = λv for some non-zero vector v. It tells you how much the matrix scales things in a particular direction.

How to find eigenvalues (2×2): Solve det(A − λI) = 0.

📝

det(A − λI) = (1−λ)² − 1 = λ² − 2λ = 0

So λ(λ − 2) = 0, giving eigenvalues λ = 0 and λ = 2.

An orthogonal matrix P satisfies P′P = I. It rotates/reflects without stretching. Every symmetric matrix can be diagonalized using an orthogonal matrix.

Section 01

What is a Quadratic Form?

Or: How to score a design composition with matrices

In statistics, we constantly work with sums of squares — total variation (TSS), residual error (RSS), group differences (SSB). All of these can be written as something called a quadratic form. Let's see what that actually means.

💡

Before we define anything formally, think about this: if you have 3 design elements with intensity values, and a client brief that says how to combine them — how would you compute a single "total impact" score? That's exactly what a quadratic form does.

Let X = (X₁, X₂, …, Xn)′ be a vector of random variables, and let A be a symmetric n × n matrix. A quadratic form is:

Q = X′AX = n Σ i=1 n Σ j=1 aijXiXj

What does this actually do? It takes your vector X (the ingredients), runs them through the matrix A (the recipe), and produces a single number Q (the result). The matrix A controls how the elements interact.

🎨

Design Analogy

Imagine you're in Illustrator, building a logo with three elements. Each element has an "intensity" value — that's your vector X. The client brief specifies how they combine: "Element 1 and Element 2 should reinforce each other, Element 3 is independent." That client brief IS the matrix A. The final design impact score — one single number that captures how the whole composition hits — that's Q = X′AX.

The Simplest Case: A = I (Identity)

When A is the identity matrix, there are no cross-terms. Each variable just gets squared:

If X₁, X₂ are independent N(0, 1), then Q = X₁² + X₂² ∼ χ²(2). Each squared standard normal adds one degree of freedom.

What the Matrix Entries Mean

Diagonal entries (a₁₁, a₂₂) produce pure squared terms like X₁².

Off-diagonal entries (a₁₂, a₂₁) create cross-product interactions like X₁X₂. These are where elements "blend" together — and they change everything about the distribution.

Idempotent Matrices: "Already Settled"

A matrix A is idempotent if A² = A. Apply the transformation once, apply it again — same result. The identity matrix I is idempotent: I² = I.

🎨

Design Analogy

You know how Flatten Layers works in Illustrator? Hit Ctrl+Shift+E, everything merges down. Hit flatten again — nothing changes. It's already flat. That's exactly what A² = A means.

E be like say — you flatten am once, e don settle. Flatten am again? Same thing. Na so A² = A dey work.

Why does this matter? Because idempotent matrices have a magical property: their eigenvalues can only be 0 or 1. No weird scaling. Each direction either contributes fully or not at all. This is the foundation of the Distribution Theorem we'll see shortly.

Key Takeaways

  • A quadratic form Q = X′AX takes a vector and a matrix and produces one number.
  • Diagonal entries of A → squared terms. Off-diagonal entries → cross-product interactions.
  • When A = I (identity), Q is simply the sum of squares: Q = ΣXᵢ².

The identity matrix was straightforward. But what happens when A has off-diagonal entries — when your design elements start interacting? That's where eigenvalues come in.

Section 02

Eigenvalues: The Secret Decoder

Breaking complex compositions into independent layers

We saw that Q = X₁² + X₂² has a nice χ² distribution. But what about Q = (X₁ + X₂)²? This expands to X₁² + 2X₁X₂ + X₂² — there's a cross-term now. How do we handle that?

🤔

Quick prediction: Q = (X₁ + X₂)² involves two variables. Do you think it follows a χ²(2) distribution?

Let's work through this step by step.

Q = (X₁ + X₂)² = X₁² + 2X₁X₂ + X₂²

Step 2 — Write the matrix:

Step 4 — Diagonalize: There exists an orthogonal P such that P′AP = diag(2, 0). Define Z = P′X. Since P is orthogonal and X has independent standard normals, Z also has independent standard normals.

Q = 2Z₁² + 0·Z₂² = 2Z₁²

Step 5 — State the distribution:

Q ∼ 2χ²(1)
⚡

Key Insight

Two variables, but only 1 degree of freedom! The eigenvalue 0 means one dimension contributes nothing — like an empty layer in your Photoshop file. The eigenvalue 2 means the remaining component is scaled by 2. The rank (number of nonzero eigenvalues) determines the true degrees of freedom, not the number of variables.

🏀

Basketball Analogy

Think about how Ant-Man breaks down a game. He scores from midrange, from three, and at the rim. Eigenvalue decomposition is like separating his total scoring into those independent categories. Each eigenvalue tells you the "point weight" of that scoring avenue. A zero eigenvalue? That's a shot type he never takes — it contributes nothing to the total.

🎨

Design Analogy

Your original design file is complex — elements overlap, interact, blend into each other. Diagonalization is like exporting each element to its own separate artboard. Now each lives independently. You can analyze each element on its own without the others interfering. That's what P′AP = D does to a matrix — untangles everything.

Section 03

The Distribution Theorem

When quadratic forms guarantee chi-square

Eigenvalues can be anything — 0, 2, 3, 17.5. But there's a special case that makes life beautiful: when the matrix A is both symmetric AND idempotent. Then the eigenvalues can only be 0 or 1. And that changes everything.

💡

If eigenvalues can only be 0 or 1, what does that mean for the quadratic form? Think about it: no scaling, no weird weights. Each independent component either contributes fully (eigenvalue = 1) or not at all (eigenvalue = 0). Clean.

The Distribution Theorem

If X ∼ Nn(0, σ²I) and A is a symmetric idempotent matrix with rank r, then:

Q = X′AX ∼ σ²χ²(r)

The Proof — Step by Step

This proof isn't a wall of symbols. It's a logical chain — each step builds cleanly on the last.

1

Decompose: Since A is symmetric, there exists an orthogonal matrix P such that P′AP is diagonal.

2

Constrain: Since A is idempotent (A² = A), the diagonal entries can only be 0 or 1. (If λ² = λ, then λ = 0 or λ = 1.)

3

Simplify: So P′AP = diag(Ir, 0) — the first r entries are 1, the rest are 0.

4

Transform: Define Y = P′X. Since P is orthogonal, Y ∼ Nn(0, σ²I). The distribution is preserved.

5

Conclude: Q = X′AX = Y′ diag(Ir, 0) Y = Y₁² + ⋯ + Yr².

Since each Yᵢ/σ ∼ N(0, 1), we get Q/σ² ∼ χ²(r). ∎

Na so e simple reach! Once A behave well — symmetric, idempotent — the distribution just dey fall in place.

🎨

Design Analogy

In a Photoshop file with 10 layers, some might be empty or just duplicates. The rank tells you how many layers have actual unique content. A rank-3 matrix in a 10×10 setting means only 3 layers are doing real work. The rest are blank. Count the non-blank layers — that's your degrees of freedom.

Common Misconception

"All quadratic forms from normal variables follow chi-square." — Not true! Only when A is symmetric AND idempotent. Otherwise, you might get a scaled chi-square or a weighted sum that isn't chi-square at all.

Key Takeaways

  • Symmetric + idempotent A → eigenvalues are only 0 and 1.
  • Rank r = count of 1-eigenvalues = degrees of freedom.
  • Q = X′AX ∼ σ²χ²(r). This is THE fundamental result.

But statistics rarely gives us just one quadratic form. What if we need to split the total sum of squares into pieces? That's Cochran's Theorem.

Section 04

Cochran's Theorem

Splitting one chi-square into independent pieces

The Distribution Theorem handles one quadratic form. But in practice — ANOVA, regression, estimation — we constantly split sums of squares into pieces. Can we guarantee those pieces are independent and each follows a chi-square distribution? That's exactly what Cochran's Theorem answers.

🤔

Think about this: you have one total sum of squares ΣXᵢ². You split it into k pieces: Q₁ + Q₂ + ⋯ + Qk. Under what conditions can we be sure the pieces don't "leak" information into each other — that they're truly independent?

Cochran's Theorem

Let X₁, …, Xn ∼ N(0, σ²), and suppose:

n Σ i=1 Xᵢ² = Q₁ + Q₂ + ⋯ + Qk

where each Qj = X′AjX, and:

  1. Each Aj is symmetric and idempotent
  2. AjAl = 0 for j ≠ l   (orthogonal projections)
  3. ΣAj = I   (partition of identity)

Then: Qj/σ² ∼ χ²(rj) independently, and r₁ + r₂ + ⋯ + rk = n.

🎨

Design Analogy — CMYK Separation

Think about CMYK printing. You take one full-color logo and separate it into four independent plates: Cyan, Magenta, Yellow, Black. Each plate captures its own color information — they don't interfere with each other. Combined, they perfectly reconstruct the original. Cochran's Theorem does the same thing with sums of squares — splits one total into independent pieces that add back to the whole.

The condition AjAl = 0 is like saying the color plates don't overlap — cyan doesn't bleed into magenta. And ΣAj = I means the plates reconstruct the full image perfectly — nothing is lost.

Key Takeaways

  • Cochran's Theorem lets us decompose one χ²(n) into k independent χ²(rj) components.
  • Three conditions: each Aj idempotent, AjAl = 0, ΣAj = I.
  • Degrees of freedom add up: r₁ + r₂ + ⋯ + rk = n.

Section 05

Sample Variance: Where Theory Meets Practice

Why we divide by n − 1 (finally, the real reason)

Cochran's Theorem sounds abstract. But it's the engine behind one of the most important results in all of statistics — the distribution of the sample variance.

💡

You've probably been told to divide by n−1 when computing sample variance. But WHY n−1 and not n? It's not a convention. It's not a correction. It's a consequence of Cochran's Theorem.

The Decomposition

Start with X₁, …, Xn ∼ N(μ, σ²), independent. The total variation around the true mean μ splits perfectly:

Σ(Xᵢ − μ)² = Σ(Xᵢ − X̄)² + n(X̄ − μ)²

This is a Cochran decomposition! Let:

  • Q₁ = Σ(Xᵢ − X̄)² — the variation within the sample
  • Q₂ = n(X̄ − μ)² — the variation of the mean itself

The matrices A₁ and A₂ are idempotent and orthogonal (A₁A₂ = 0). By Cochran's Theorem:

Q₁/σ² ∼ χ²(n−1)

n−1 degrees of freedom for within-sample variation

Q₂/σ² ∼ χ²(1)

1 degree of freedom for the mean

They are independent. And (n−1) + 1 = n. ✓

⭐

The Big Result

(n−1)S²σ² = Σ(Xᵢ − X̄)²σ² ∼ χ²(n−1)

This is the foundational result for confidence intervals and hypothesis tests. And as a bonus: X̄ and S² are independent!

🏀

Basketball Analogy

Picture a starting five. The team's scoring average is 20 points per player. If you know what four players scored, the fifth player's score is already locked in — it's whatever makes the average hit 20. Out of 5 scores, only 4 are "free." One degree of freedom gets consumed by the average. That's why we divide by n−1, not n.

Key Takeaways

  • Total variation splits into within-variation (n−1 df) and mean-variation (1 df).
  • (n−1)S²/σ² ∼ χ²(n−1) — the foundation for inference.
  • X̄ and S² are independent — a direct consequence of Cochran's Theorem.

When Weights Break Chi-Square

Not every quadratic form plays nice

We've been living in a world where A is idempotent — eigenvalues are 0 or 1 — and everything falls neatly into chi-square distributions. But what about Q = X₁² + 2X₂² + 3X₃²? The matrix is diagonal with entries 1, 2, 3. Not idempotent. Not equal weights.

🤔

Gut check: Q = X₁² + 2X₂² + 3X₃² involves three chi-square(1) variables. Is Q itself a chi-square distribution?

The Setup

Here, A is diagonal:

A =
1
0
0
0
2
0
0
0

So Q = 1·X₁² + 2·X₂² + 3·X₃². Each term is a scaled χ²(1). But scaling breaks the chi-square property — c·Z is NOT chi-square unless c = 1.

Proving It's Not Chi-Square

If Q were χ²(r), it would satisfy E(Q) = r and Var(Q) = 2r, so Var(Q) = 2·E(Q). Let's check:

E(Q) = 1·E(X₁²) + 2·E(X₂²) + 3·E(X₃²) = 1 + 2 + 3 = 6

Var(Q) = Var(1·X₁²) + Var(2·X₂²) + Var(3·X₃²)

Since Var(c·χ²(1)) = c²·Var(χ²(1)) = c²·2 = 2c²:

Var(Q) = 2(1² + 2² + 3²) = 2(1 + 4 + 9) = 28

❌

Chi-square check: Is Var(Q) = 2·E(Q)? Is 28 = 12? NO!

If Q ∼ χ²(r), we'd need r = 6 (from E) but also 2r = 28 → r = 14. Contradiction. Q is not chi-square.

The General Formulas

For any weighted sum Q = a₁X₁² + a₂X₂² + ⋯ + anXn²:

E(Q) = Σaᵢ = tr(A)
Var(Q) = 2Σaᵢ²
🔑

Quick chi-square check: Q ∼ χ²(r) only if Var(Q) = 2·E(Q), which requires all weights to equal 1.

🏀

Basketball Analogy

If Ant scored ONLY from free throws (1 point each), his total would follow a clean chi-square pattern — each shot is equally weighted. But real Ant mixes free throws (1pt), midrange (2pt), and threes (3pt). Each shot type carries a different weight. When the weights aren't all equal, the total distribution isn't a simple chi-square — it's a weighted combination with its own unique shape.

Omo, this one no follow chi-square o. The different weights don scatter the pattern.

Key Takeaways

  • Q = Σaᵢχ²(1) is NOT chi-square unless all aᵢ = 1.
  • E(Q) = Σaᵢ (sum of weights). Var(Q) = 2Σaᵢ² (twice sum of squared weights).
  • Quick check: Q ∼ χ²(r) only if Var(Q) = 2·E(Q).

Practice & Synthesis

Putting it all together

You've learned the definition, the eigenvalue trick, the Distribution Theorem, Cochran's Theorem, the sample variance result, and the weighted sum edge case. Now let's bring it all together.

The Decision Framework

When you encounter a quadratic form, follow these steps:

1

Write Q = X′AX. Identify the matrix A.

2

Is A = I? → Q ∼ χ²(n). Done.

3

Is A idempotent? → Find rank r → Q ∼ σ²χ²(r). Done.

4

Is A diagonal with unequal entries? → Weighted sum, NOT chi-square. Find E(Q) and Var(Q).

5

None of the above? → Diagonalize. Find eigenvalues. Q = ΣλᵢZᵢ².

Glossary

Quadratic Form

An expression Q = X′AX where X is a vector of random variables and A is a symmetric matrix. Produces a single number from the vector.

Symmetric Matrix

A matrix where A = A′ (equals its own transpose). Entries mirror across the main diagonal.

Idempotent Matrix

A matrix where A² = A. Applying it twice gives the same result as applying it once. Its eigenvalues can only be 0 or 1.

Eigenvalue

A scalar λ satisfying Av = λv for some non-zero vector v. Eigenvalues reveal how much a matrix scales in each independent direction.

Orthogonal Matrix

A matrix P where P′P = I. Orthogonal transformations rotate/reflect without stretching, preserving distances and distributions.

Diagonalization

The process of finding P such that P′AP is diagonal. This separates a matrix into independent components along its eigenvectors.

Rank

The number of nonzero eigenvalues of a matrix. Equals the effective number of independent dimensions in the quadratic form.

Chi-Square Distribution

The distribution of a sum of squared standard normal variables. χ²(k) has mean k and variance 2k.

Degrees of Freedom

The number of independent components in a sum of squares. For quadratic forms with idempotent A: df = rank(A).

Cochran's Theorem

States conditions under which a sum of squares can be decomposed into independent chi-square components: each matrix idempotent, mutually orthogonal, summing to the identity.

Distribution Theorem

If A is symmetric idempotent with rank r and X ∼ N(0, σ²I), then X′AX ∼ σ²χ²(r).

Sample Variance

S² = Σ(Xᵢ − X̄)²/(n−1). By Cochran's Theorem, (n−1)S²/σ² ∼ χ²(n−1).

Weighted Chi-Square Sum

Q = Σaᵢχ²(1) where weights aᵢ are not all equal. NOT a chi-square distribution. E(Q) = Σaᵢ, Var(Q) = 2Σaᵢ².

Statistical Independence

Two quadratic forms are independent if their projection matrices satisfy AⱼAₗ = 0 — they project onto orthogonal subspaces.

Projection Matrix

A matrix that is both symmetric and idempotent. Projects vectors onto a subspace. Used in Cochran's Theorem decompositions.
